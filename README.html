Chapter 1

📘 Chapter 1. 📚 HTML 태그 총정리

1. 🏗️ 문서 구조 태그
태그
설명
<!DOCTYPE>
문서 타입 선언 (HTML5 기준: <!DOCTYPE html>)
<html>
HTML 문서의 루트
<head>
문서의 메타 정보 영역
<meta>
문자셋, 설명, 키워드 등 설정
<title>
문서 제목 (탭에 표시됨)
<body>
화면에 보이는 콘텐츠 부분

2. 📝 텍스트 콘텐츠 태그
태그
설명
<h1> ~ <h6>
제목 (1이 가장 크고 6이 가장 작음)
<p>
문단
<br>
줄바꿈 (break line)
<hr>
수평선 (구분선)
<strong>
굵은 글씨 (의미 강조)
<b>
굵은 글씨 (단순 스타일)
<em>
기울임 (의미 강조)
<i>
기울임 (단순 스타일)
<mark>
형광펜 표시
<blockquote>
인용 문단
<code>
코드 블록
<pre>
서식을 그대로 유지한 텍스트

3. 🔗 링크 및 미디어 태그
태그
설명
<a href="url">
하이퍼링크 (anchor)
<img src="경로">
이미지
<audio>
오디오 삽입
<video>
비디오 삽입
<source>
오디오/비디오 파일 경로
<iframe>
다른 페이지를 삽입

4. 🧱 레이아웃/구조 태그 (HTML5 시멘틱 태그)
태그
설명
<div>
구역을 나눌 때 사용하는 기본 컨테이너
<span>
텍스트 일부에 스타일을 적용할 때 사용
<header>
머리글 영역
<nav>
내비게이션 링크 영역
<main>
본문 주요 콘텐츠
<section>
논리적 구획 (섹션)
<article>
독립적인 기사 콘텐츠
<aside>
사이드 콘텐츠 (광고, 링크 등)
<footer>
꼬리말 영역

5. 📋 목록 태그
태그
설명
<ul>
순서 없는 목록 (bullet)
<ol>
순서 있는 목록 (1, 2, 3)
<li>
리스트 항목 (ul/ol 내부에서 사용)
<dl>
정의 목록
<dt>
정의 목록 제목
<dd>
정의 목록 설명

6. 🧾 테이블 태그
태그
설명
<table>
표 생성
<tr>
행 (table row)
<td>
셀 (table data)
<th>
셀 (table header)
<thead>
표의 머리글 그룹
<tbody>
본문 그룹
<tfoot>
표의 바닥글 그룹
<caption>
표 제목
<col>, <colgroup>
열 스타일 지정

7. 🧠 폼(Form) 및 입력 태그
태그
설명
<form>
폼 전체를 감싸는 태그
<input>
사용자 입력 필드
<textarea>
여러 줄 입력 필드
<select>
드롭다운 메뉴
<option>
선택 항목
<label>
입력 필드의 이름 지정
<button>
버튼 생성
<fieldset>
폼 영역 구분
<legend>
필드셋 제목
<datalist>
자동완성 목록
<output>
계산 결과 등 출력 표시

<input> 태그 속성에 따라 다양한 종류 가능:
type="text", "password", "checkbox", "radio", "submit", "reset", "file", "email", "date" 등

8. ⚙️ 스크립트 및 스타일 태그
태그
설명
<style>
문서 내 CSS 작성
<link>
외부 CSS 파일 연결
<script>
자바스크립트 작성
<noscript>
JS 미지원 시 대체 내용

9. 🧼 의미 없는 태그 (비시멘틱)
태그
설명
<div>
의미 없는 컨테이너
<span>
의미 없는 인라인 요소 (스타일용)

10. 📦 기타 특수 태그
태그
설명
<template>
자바스크립트에서 사용하는 HTML 조각 템플릿
<canvas>
자바스크립트로 그래픽 그리기
<svg>
벡터 그래픽 정의
<details>
토글 가능한 섹션
<summary>
details 태그 요약 제목
<dialog>
대화상자 모달 창

✅ 부록: 태그 속성의 예
모든 HTML 태그는 속성(attribute)을 가질 수 있습니다.
<a href="https://naver.com" target="_blank">네이버로 이동</a>

href = 이동할 주소
target="_blank" = 새 창으로 열기
